FANCY GALLERY

Galeria dinámica usando la API de https://unsplash.com, llamada AJAX con Jquery , carga de imágenes al hacer scroll down de 10 en 10. CSS con Bootstrap, preparado para gestionar las imágenes en responsive para diferentes tipos de pantalla con la libreria Masonry para gestionar el posicionamiento de las fotos, Imágenes con lazyload de carga previa con posibilidad de ampliar cada una con una galería de imágenes creada con la librería Fancybox, cambiada la configuración de X-frame para que te permita acceder a la imagen en grande al clickar en la foto con la extension para google Chrome IGNORE X-FRAME (https://chrome.google.com/webstore/detail/ignore-x-frame-headers/gleekbfjekiniecknbkamfmkohkpodhe?). Dos tipos diferentes de google fonts combinadas para mejor resultado en la parte superior para titulo (Rubik) y para este texto (Roboto).
